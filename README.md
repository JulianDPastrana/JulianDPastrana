## About Me

Machine learning engineer and data scientist who builds machine learning models and LLM-powered applications (OpenAI, Anthropic, and Gemini APIs). End-to-end experience in Python for time-series forecasting, probabilistic modeling, and signal processing. Currently pursuing doctoral studies in Engineering at the **Grupo de Investigación en Automática** (Minciencias category A1), Universidad Tecnológica de Pereira.
